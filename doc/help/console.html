Console.exe - представляет собой программу обработки изображений, воспринимающую команды вводимые с клавиатуры.

При запуске из командной строки можно использовать следующие параметры, перечисляемые в произвольном порядке:

  • -f имя_файла_сценария - для запуска обработки пакетного файла обозначенного путем имя_файла_сценария
  • -l имя_log_файла - для переопределения файла для вывода результатов. По умолчанию используется файл fractals.log.html

Обработка нажатий с клавиатуры производится на основе сценария, прописанного в настройках как "Console Input" (по-умолчанию это AutoScripts/Console.sh)

Подробнее о работе в пакетном режиме смотрите в разделе Пакетная обработка
