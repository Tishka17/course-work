Fractals Analyzer имеет встроенный интерпретатор скриптового языка во много аналогичного языкам используемым в ОС Linux в качестве скриптов. Но это сходство не полное, так как его возможности пока очень малы.

Основными командами являются:

  1. if (условие) команда
    else команда
  2. echo сообщение
  3. call имя_файла
  4. goto имя_метки
  5. set /параметры имя_переменной = значение
