"Fractals analyzer"
  • Коротко
  • Описание бинарных файлов
    • Exe файлы:
      • console.exe - позволяет производить расчеты вводя команды с клавиатуры, а так же в пакетном режиме.
      • gui_.exe - представляет собой программу с графическим интерфейсом, что позволяет увидеть на экране некоторые результаты работы (само изображение, графики и разные фигуры характеризующие изображение).
    • DLL библиотеки:
      • readdata.dll - дает возможность загрузки изображений из файлов (пока только в формате bmp).
      • graphics_.dll - содержит функции по отрисовке картинки на экране.
      • approximation.dll - содержит методы апроксимации и экстраполяции
      • input.dll - необходима для организации диалогов ввода данных.
    • Стандартные плагины:
      • Filter - даёт возможности для фильтрации картинки и наложения разных эффектов
      • Genrator - генерирует некоторые стандартные фракталы
      • RandomWalk - генерирует случайные блуждания (моделирование процессов диффузии)
      • Dimension - расчитывает размерности фрактала.
  • Описание вспомогательных файлов
    • settings.ini - файл настроек
    • язаковые файлы
    • Log-файл - описание результатов работы программы
  • Руководства
    • Создание плагинов
    • Пакетная обработка файлов
